1. Tech stack

• Frontend: Next.js (React, TypeScript, App Router)
• UI: Tailwind CSS + shadcn/ui (or similar)
• Backend: Next.js API routes
• Database: Supabase (PostgreSQL + Auth)
• Auth: Supabase Auth (email/password, OAuth later)
• Realtime: Supabase channels (for live prices/signals later)
• Deployment: Vercel (frontend + API) or Replit hosting
• AI: OpenAI / Anthropic API (for research, scoring, insights)
• PWA: Next-PWA or custom service worker

3. Core modules & features (A–Z)

3.1 Auth & user system

• Email/password login
• Basic profile page
• Plan field for future monetization


---

3.2 Home dashboard

Sections:

• Market Overview: BTC, ETH, total market cap, BTC dominance, Fear & Greed, gas fees, funding rates
• Trending Narratives: list with momentum score, 24h/7d perf, top 3 tokens
• Top Movers: gainers, losers, volume, most searched
• Whale Activity: large buys/sells (placeholder or API)
• AI Insights: daily AI summary (top narratives, risks, opportunities)
• News Highlights: 3–5 headlines with summaries


---

3.3 Token search & research page

When user searches a coin:

Sections:

• Header: price, % change, market cap, volume, FDV, supply
• Score circles: Overall, Fundamental, Technical, Sentiment, Risk, Narrative Momentum
• Final Grade: A+ / A / B / C / D / F + explanation
• Overview: what it does, use cases, value prop
• Tokenomics: distribution, unlocks, vesting
• On‑chain metrics: holders, tx count, TVL (if DeFi)
• Social metrics: Twitter, Telegram, Discord, growth
• Developer activity: GitHub commits, repos (if available)
• Narrative context: which narratives it belongs to, momentum
• Whale activity: accumulation/distribution (if available)
• News related to token
• AI Summary: strengths, weaknesses, risks, opportunity level


AI endpoint:

• Input: token data JSON + market data + narrative data
• Output: scores, grade, explanations, summary


---

3.4 Narratives page

• List of all narratives (DePIN, AI, L2, DeFi, Meme, Gaming, RWA, etc.)
• Each card: icon, description, 24h/7d perf, momentum score
• Click → narrative detail page:• Description
• Momentum history
• Top tokens
• AI commentary



---

3.5 Signals module

• Signals list: token, action, entry, target, confidence, % to target, status
• Auto‑update job (cron or scheduled function):• Fetch prices
• If price ≥ target → status = hit_target
• If now > expires_at → status = expired

• UI:• Show only active by default
• Filters: active, hit, expired
• Progress bar: % to target



---

3.6 Portfolio module

Features:

• Manual add:• Token, amount, buy price, date, target price

• Portfolio analytics:• Total value
• Total invested
• Total PnL (USD + %)
• Allocation by token
• Allocation by sector/narrative
• Portfolio risk score (weighted)
• Portfolio momentum score (weighted)
• Portfolio target progress (weighted % to targets)

• Per‑position:• Value, PnL, target price, % to target, progress bar

• Future:• Exchange connections (Binance, Bybit, etc.)
• Wallet connections (EVM, Solana, etc.)
